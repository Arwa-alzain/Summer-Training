# Task 01 – Personal Portfolio Website

## 📌 Overview

This project is a responsive personal portfolio website developed using **HTML5** and **CSS3**. It serves as a professional online portfolio that showcases my background, technical skills, projects, certifications, and resume.

The website was designed to provide visitors with an overview of my work and make it easy to access my projects and professional profiles.

---

## 🔗 Live Demo

Visit the website here:

**https://arwa-alzain.github.io/Portfolio/**

---

## 🎯 Objectives

- Design a professional personal portfolio website.
- Showcase my technical skills and projects.
- Display certificates and achievements.
- Provide online access to my resume.
- Allow visitors to easily contact me through multiple platforms.
- Practice front-end web development using HTML and CSS.
